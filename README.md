# Discount Calculator

A simple and beginner-friendly Discount Calculator built using **HTML, CSS, and JavaScript**.  
This tool allows users to enter the original price and discount percentage.  
It then calculates:

- Final Price after discount  
- Total Savings  

The project uses only **basic JavaScript operators** making it perfect for beginners learning logic building.

## Features
- User-friendly interface  
- Real-time discount calculation  
- Responsive and clean design  
- Uses JavaScript arithmetic operators  

## Technologies Used
- HTML  
- CSS  
- JavaScript
